FavIcon

symbols or images that represent a website.

favicon are small icons that appear in the browser tab and bookmarks bar.

How to Create a FavIcon

We use link tag in the head section of HTML documents

Attributes of FavIcon

  • rel: specifies the relationship between the current document and the linked resource. For favicons, it is set to "shortcut icon".
  • href: specifies the URL of the favicon image.
  • type: specifies the MIME type of the linked resource, which is typically "image/x-icon" for favicons.
