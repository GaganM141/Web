HTML

html is used to build a structure of webpage.

hyper(h)-->reference

text(t)-->content to be displayed in user interface(text, image, audio, etc)

markup(m)-->pre defined set of rules or formats

language(l)-->medium to communicate

html is case insensitive.

.html extension is used to create a html file.

Tags

Predefined keywords enclosed in angular brackets.

types

1. paired tags

paired tags has 2 parts opening and closing tag

ex: html

2. unpaired tags

unpaired tag is a self colsing or single tag

ex: br, img

Elements

Combination of opening tag, closing tag along with content

Types

1. Block level elements

it occupies complete view port width or parent width

it always starts in new line

ex: all heading tags

2. Inline level tags

only occupies content width

doesn't start in new line

ex: bold, span
