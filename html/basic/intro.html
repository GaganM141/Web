preLoad attribute

preload attribute is used to specify how the video should be loaded when the page loads.

It accepts three values: none, metadata, and auto.

auto completes the loading of the video as soon as the page is loaded.

metadata only loads the metadata of the video, such as duration and dimensions, but not the actual video data.

none does not load the video at all until the user clicks the play button.

It is recommended to use the 'metadata' value for the preload attribute. as it only loads few information about the video.

We almost use 'auto' value because a website may contain many videos and if auto is given it trys to load all videos at a time which effects performance.

ex: movies

auto is the default value for the preload attribute.
